# タグ改善・加点改善のルール

## 改善候補

- 新規タグ候補
- 既存タグの意味整理
- 過剰加点タグ
- 減点が必要な条件
- 条件付きで有効なタグ
- 4コマ漫画化できる学び

## 例

- 芝戻りは有効だが、前走ダートから大幅ペースアップする逃げ先行馬は注意
- 度外視可能は買いサインではなく再評価サイン
- 血統加点は馬場・距離・展開とセット
- A判定でも馬場逆風なら投資額を下げる
- 人気薄でもオッズが足りなければ妙味なし
- 好走実績は現在条件に再接続できるかを見る
- 高速馬場で追走負荷が大きい馬は、次走ペース緩和で見直し
- 差し届く馬場で逃げて失速した馬は、展開不向きとして扱う余地あり

## 改善案の強さ

- 強：複数事例で確認
- 中：似た事例が複数ある
- 弱：1例のみ
- メモ：仮説段階

## 本体反映ルール

- このスキルは改善案を出すだけ
- app.pyやAI本体コードは変更しない
- 実装はユーザーが別途判断する
- 変更する場合は必ずバックアップ・コミット後に行う
