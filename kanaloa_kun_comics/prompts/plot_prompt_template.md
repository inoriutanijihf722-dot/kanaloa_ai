# 4コマ台本作成プロンプトテンプレート

以下の入力をもとに、X投稿用の「カナロアAI 4コマ漫画」の台本を作成してください。

## 入力欄

```text
今回のテーマ：
伝えたい教訓：
登場させたい競馬AIワード：
オチの方向性：
ギャグ強度：
```

## 前提

登場キャラは3人です。

- カナロア君：栗毛のかわいい馬AIマスコット。期待値・資金管理・冷静な判断を伝える。穏やかだがツッコミは鋭い。
- 慎重な分析者：メガネの若い男性。真面目で少し気弱。AI診断を読み上げ、読者目線でツッコむ。
- 大胆な先輩：派手な服の夢見るギャンブラー。テンションが高く、すぐ勝った気になる。ボケ担当。

## 構成ルール

1. 1コマ目：AI判定や状況で期待が高まる
2. 2コマ目：先輩が感情で暴走する
3. 3コマ目：カナロア君が本質を冷静に指摘する
4. 4コマ目：現実的な行動・教訓で着地する

## セリフルール

- セリフは短く、画像内で読みやすい長さにする
- 1コマあたり吹き出しは2〜3個まで
- 先輩は感情的で大げさにする
- 分析者は読者目線で慌てる
- カナロア君は冷静に本質を言う
- ギャンブルを過度に煽らない
- 最後は「期待値」「資金管理」「継続」のどれかが残るようにする

```text
状況：
セリフ：

```

## 生成時のお願い

- 初見でも意味が分かる4コマにしてください
- 2コマ目は大げさで面白くしてください
- 3コマ目は保存したくなる正論にしてください
- 4コマ目はかわいく、前向きに着地してください
