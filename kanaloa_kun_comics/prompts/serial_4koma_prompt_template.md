# 連載4コマ台本作成プロンプトテンプレート

以下の入力と参照ファイルをもとに、カナロアAI 4コマ漫画の連載台本を作成してください。

## 入力欄

```text
episode_no：
今回のテーマ：
今回の教訓：
入れたい競馬AIワード：
ギャグ強度：
使用禁止にしたい過去ネタ：
先輩の成長段階：
カナロア君の締めセリフ候補：
```

## 参照するもの

- `docs/serial_bible.md`
- `docs/senpai_growth_arc.md`
- `docs/character_state_rules.md`
- `docs/episode_registry_guide.md`
- `data/episode_registry.csv`
- `data/character_state.json`

## 必須ルール

- 前回と同じテーマ、同じオチ、同じ構図にしない
- 先輩は現在phaseに応じた理解度にする
- 先輩を急に成長させすぎない
- ただし毎回1つだけ小さく学ばせる
- カナロア君は冷静な学びを担当
- 分析者は読者目線のリアクション担当
- 最後はギャンブル煽りではなく投資家思考に着地する
- 「絶対勝てる」「必ず儲かる」は禁止
- 画像生成前に日本語セリフを短く整える

## 出力形式

### 1. episode metadata

```text
episode_no:
phase:
theme:
title:
subtitle:
lesson:
```

### 2. 先輩の今回の誤解

先輩が今回どこを感情的に誤読したかを1〜2文で書く。

### 3. 今回の学び

この回で先輩が持ち帰る学びを1つだけ書く。

### 4. 4コマ台本

```text
1コマ目：
状況：
セリフ：
- 分析者「」
- 先輩「」
- カナロア君「」

2コマ目：
状況：
セリフ：
- 先輩「」
- 分析者「」

3コマ目：
状況：
セリフ：
- カナロア君「」
- 分析者「」
- 先輩「」

4コマ目：
状況：
セリフ：
- 先輩「」
- 分析者「」
- カナロア君「」
```

### 5. 画像生成AI用プロンプト

縦型4コマ漫画として、キャラクター外見、各コマの状況、短い日本語セリフ、読みやすい吹き出し、禁止事項を含める。

### 6. X投稿文3案

- 短め
- 標準
- 学び強め

### 7. episode_registry.csv 追記用1行

CSVの列順に合わせて1行で出力する。

### 8. character_state.json 更新案

先輩のパラメータ、known_lessons、growth_notes の更新案をJSON断片で出力する。

### 9. 前作流用チェック

```text
直近3話とのテーマ重複：
直近3話とのオチ重複：
直近3話との構図重複：
再利用を避けた要素：
問題があれば修正案：
```
