# Eカナロア診断入力補助β 手順書

## 目的

人気ランクEかつ父または母父にロードカナロア/カナロアを持つ馬を、【Eカナロア母集団】候補として確認し、Kanaloa AI Pro Ver13.5 の診断入力フォームへ安全に転記するための手順です。

この手順は母集団抽出と診断入力補助だけを扱います。AI診断実行、保存、CSV追記、【ディープバリュー候補】付与は自動化しません。

## 対象馬

- 人気ランクがE
- 父または母父にロードカナロア/カナロアが確認できる
- 人間が【Eカナロア母集団】候補として確認した馬

## 入力するスクショ情報

スクショやOCR結果から、分かる範囲で以下を貼り付けます。

- 馬名
- 馬主
- 性齢
- 人気ランク
- 単勝オッズ
- 騎手
- 厩舎
- 父
- 母父
- 備考

## Eカナロア抽出βの使い方

1. AI診断・入力タブを開く。
2. 「人気ランクEカナロア抽出β」を開く。
3. OCR結果貼り付け欄にスクショ由来テキストを貼る。
4. 必要なら手動修正欄で補足する。
5. 抽出テーブルで馬名、人気ランク、父、母父、単勝オッズを確認する。

## 確認済みチェック

内容を目視確認した候補だけ、確認済みにチェックします。

確認済みが0頭の場合、馬1フォームへの下書き反映ボタンは表示されません。確認済みが2頭以上の場合も、誤反映防止のため下書き反映ボタンは表示されません。

## 診断入力用サマリー確認

確認済み候補には、診断入力用サマリーが表示されます。

- 馬名
- 馬主
- タイプ
- 性別
- 騎手
- 厩舎
- 母父(父)
- 人気
- 単勝
- 備考

## 馬1フォームへ下書き反映

確認済み候補が1頭だけの場合に限り、「馬1フォームへ下書き反映」ボタンが表示されます。

ボタンを押すと、空欄でない項目だけが馬1フォームへ反映されます。反映先は馬1のみです。

- 馬名 -> n_0
- 馬主 -> ow_0
- タイプ -> t_0
- 性別 -> s_0
- 騎手 -> j_0
- 厩舎 -> tr_0
- 母父(父) -> mf_0
- 人気 -> r_0
- 単勝 -> o_0

## ユーザー目視確認

下書き反映後も、必ず馬1フォームを目視確認してからAI診断してください。

特に、複勝、枠、テンP、上がりP、成長サイン、映像評価は自動補完しません。必要に応じて人間が入力します。

## AI診断実行と保存

AI診断実行はユーザーが手動で押します。保存もユーザーが手動で行います。

このβは `kanaloa_investment_log.csv` に自動追記しません。既存CSV保存処理にも接続しません。

## ディープバリュー候補

【ディープバリュー候補】は自動付与しません。

【Eカナロア母集団】候補は研究用の母集団タグであり、買い推奨やディープバリュー判定ではありません。
