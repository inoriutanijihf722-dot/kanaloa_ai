# 次の10話分 テーマ候補作成プロンプト

以下の参照ファイルを確認し、カナロアAI 4コマ漫画の次の10話分のテーマ候補を作成してください。

## 参照するもの

- `docs/serial_bible.md`
- `docs/senpai_growth_arc.md`
- `docs/character_state_rules.md`
- `data/episode_registry.csv`
- `data/character_state.json`

## 入力欄

```text
開始episode_no：
現在の先輩phase：
直近で避けたいテーマ：
強めたい学び：
使いたい競馬AIワード：
```

## テーマバランス

以下をシリーズ全体でバランスよく含めてください。

- A判定
- B判定
- C判定
- 度外視可能
- 次走狙い
- 見送り
- オッズ
- 資金管理
- 馬場バイアス
- 展開
- 距離変更
- 血統
- 騎手
- 回収率
- 的中率
- 連敗時のメンタル
- 取り返し馬券禁止
- 検証ログ

## 作成ルール

- 直近3話と同じテーマ、同じオチ、同じ演出にしない
- episode_no に応じて先輩の phase を自然に進める
- 各話で先輩が学ぶことは1つだけにする
- 先輩を急に完璧な投資家にしない
- ギャグ要素を残しつつ、最後は期待値・資金管理・検証に着地させる

## 出力形式

各話を以下の形式で出力してください。

```text
episode_no:
phase:
theme:
title案:
先輩の誤解:
学び:
カナロア君の締めセリフ案:
前回との差別化ポイント:
```
