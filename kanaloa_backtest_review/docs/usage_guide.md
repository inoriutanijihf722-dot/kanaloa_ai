# 使い方

## 手順

1. 過去レースをカナロアAIで診断する
2. 診断結果、適用要素、判定、期待値を保存する
3. 実際の結果とレース内容を入力する
4. backtest_review_prompt.md を使って検証メモを作る
5. 改善案、タグ案、4コマネタを抽出する
6. 必要ならX投稿用に要約する
7. 予想実績ではなく検証メモとして保存する

## 入力として渡すとよい情報

- レース名
- 馬名
- AI判定
- 期待値
- 推奨投資額
- 適用要素
- オッズ
- 人気
- 着順
- 着差
- 通過順
- 上がり
- 馬場
- 展開
- 不利
- レース後メモ
- スクショ
- 馬場バイアスレポートがある場合はその要約

## 出力の使い道

- 自分用検証メモ
- X投稿用の透明性ある検証メモ
- 4コマ漫画テーマ
- タグ改善メモ
- 将来のAIロジック改善候補
