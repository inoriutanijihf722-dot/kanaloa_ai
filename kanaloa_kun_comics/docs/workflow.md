# 制作ワークフロー

## 目的

毎回テーマを変えるだけで、X投稿用の「カナロアAI 4コマ漫画」を安定して作るための手順です。

## 手順

### 1. テーマを決める

`ideas/comic_topics.md` からテーマを選ぶか、新しいテーマを1つ決めます。

例：

- A判定の破壊力
- 見送りこそ投資
- オッズを見て先輩が暴走する
- 勝つことより続けること

### 2. 台本を作る

`prompts/plot_prompt_template.md` を使って、4コマ台本を作ります。

入力するもの：

- 今回のテーマ
- 伝えたい教訓
- 登場させたい競馬AIワード
- オチの方向性
- ギャグ強度

### 3. セリフを短く整える

画像生成AIは長い日本語を苦手にすることがあるため、セリフを短く整えます。

目安：

- 1吹き出しは1〜2行
- 1コマあたり吹き出しは2〜3個まで
- 3コマ目の正論は短く分ける
- 4コマ目の教訓は一言で締める

### 4. 画像生成AI用プロンプトにする

`prompts/image_prompt_template.md` を使って、画像生成AIに渡す完成プロンプトを作ります。

必ず入れるもの：

- カナロア君の見た目
- 分析者の見た目
- 先輩の見た目
- 縦型4コマ
- 各コマの状況
- 各コマの固定セリフ
- 禁止事項
- 読みやすさ重視

### 5. 画像生成AIで作る

画像生成AIにプロンプトを渡し、縦型4コマ画像を生成します。

生成時の注意：

- 1回目から完璧を狙いすぎない
- 文字が崩れた場合は、セリフをさらに短くする
- 構図が弱い場合は、各コマの主役を明記する
- カナロア君が似ていない場合は、参考画像の特徴を再指定する

### 6. QAチェックする

`docs/qa_checklist.md` を使って確認します。

特に重要：

- カナロア君の特徴
- 文字の読みやすさ
- 2コマ目の暴走感
- 3コマ目の学び
- ギャンブル煽りになりすぎていないか

### 7. 必要なら修正指示を作る

修正指示は、原因を1つずつ分けて書きます。

例：

- 「カナロア君を参考画像にもっと近づけてください」
- 「2コマ目の先輩をもっと大げさにしてください」
- 「3コマ目の文字を減らして、吹き出しを大きくしてください」
- 「図解風に見えるので、完成した漫画イラスト風にしてください」

### 8. X投稿文を作る

`prompts/post_prompt_template.md` を使って、投稿文を3パターン作ります。

選ぶ基準：

- 短め：画像のインパクトを優先したいとき
- 標準：もっとも使いやすい基本形
- 少し熱量あり：テーマの学びを強く伝えたいとき

### 9. outputsに保存する

完成物は `outputs/` に保存します。

推奨ファイル名：

```text
YYYYMMDD_theme_slug.png
YYYYMMDD_theme_slug_prompt.md
YYYYMMDD_theme_slug_post.md
```

例：

```text
20260425_a_hantei_no_hakairyoku.png
20260425_a_hantei_no_hakairyoku_prompt.md
20260425_a_hantei_no_hakairyoku_post.md
```

## 毎回の最短運用

1. `ideas/comic_topics.md` からテーマを選ぶ
2. `plot_prompt_template.md` に入力する
3. 出た台本を短くする
4. `image_prompt_template.md` に台本を入れる
5. 画像生成
6. `qa_checklist.md` で確認
7. `post_prompt_template.md` で投稿文作成
8. `outputs/` に保存
