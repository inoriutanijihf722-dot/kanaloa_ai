# 4コマ品質チェックリスト

完成した4コマ、または画像生成AIに渡す直前のプロンプトを確認するためのチェックリストです。

## キャラクター

- [ ] カナロア君が参考画像の特徴を保っている
- [ ] カナロア君に栗毛、大きな青緑の目、青いスカーフ、顔の発光ライン、タブレットがある
- [ ] カナロア君が知的で冷静な存在として見える
- [ ] 慎重な分析者がメガネ、真面目、少し気弱な読者目線キャラになっている
- [ ] 大胆な先輩が派手で表情豊かに暴走している
- [ ] 3人の役割が一目で分かる

## 4コマの流れ

- [ ] 1コマ目でテーマと状況が分かる
- [ ] 1コマ目に期待感や引きがある
- [ ] 2コマ目で先輩がちゃんと暴走している
- [ ] 2コマ目がギャグとして一番大きく見える
- [ ] 3コマ目でカナロア君が冷静に本質を指摘している
- [ ] 4コマ目で現実的な行動に着地している
- [ ] 4コマ全体の流れが初見でも分かりやすい

## 学びと思想

- [ ] カナロア君の一言に学びがある
- [ ] 「期待値」の思想が残っている
- [ ] 「資金管理」の思想が残っている
- [ ] 「継続」の思想が残っている
- [ ] 感情で買いすぎないメッセージになっている
- [ ] ギャンブル煽りになりすぎていない
- [ ] 勝つことだけでなく、続けることの大切さが伝わる

## 視認性

- [ ] 文字がスマホで読める
- [ ] 吹き出しに余白がある
- [ ] セリフが長すぎない
- [ ] 重要語が見やすい
- [ ] キャラの顔と表情が潰れていない
- [ ] コマの境界がはっきりしている
- [ ] タイトルが読みやすい

## 画像生成プロンプト確認

- [ ] 縦型4コマであることが明記されている
- [ ] SNS投稿向けであることが明記されている
- [ ] 漫画風、かわいいデフォルメ、完成イラスト風が明記されている
- [ ] 棒人間禁止、図解風禁止、スライド資料風禁止が明記されている
- [ ] 3人の見た目が明記されている
- [ ] 各コマの状況が具体的に書かれている
- [ ] 各コマの固定セリフが書かれている
- [ ] 前回画像を使わない場合は、構図を新しくする指示が入っている

## X投稿向け

- [ ] 1枚画像としてインパクトがある
- [ ] 2コマ目が目を引く
- [ ] 4コマ目に保存・共感される一言がある
- [ ] 投稿文に期待値、資金管理、感情を抑える、続けることの思想が入っている
- [ ] ハッシュタグが多すぎない
