# 4コマ構成テンプレート

## 黄金パターン

1. 1コマ目：AI判定や状況で期待が高まる
2. 2コマ目：先輩が感情で暴走する
3. 3コマ目：カナロア君が本質を冷静に指摘する
4. 4コマ目：現実的な行動・教訓で着地する

この流れを守ると、毎回テーマを変えても「笑えるけれど理にかなっている」4コマになります。

## 1コマ目：期待が高まる

### 役割

- テーマを提示する
- 読者に「何が起きたか」を一瞬で伝える
- 先輩が暴走する火種を置く
- AI診断やレース状況を使って引きを作る

### よく使う演出

- タブレットに診断結果
- 大きな「A判定」「妙味あり」「見送り推奨」など
- 分析者の驚き顔
- 先輩が横から食いつく
- カナロア君は落ち着いて一言

### セリフの型

- 分析者「〇〇……A判定です！」
- 分析者「この条件、かなり期待値があります！」
- 先輩「来たァーーー！！」
- 先輩「勝ったな！！」
- カナロア君「まだ発走前です。」
- カナロア君「判定は入口です。」

### NG例

- いきなり長い説明を入れる
- 競馬用語を多くしすぎる
- 先輩がまだ何も起きていないのに落ち着きすぎている
- カナロア君が最初から説教しすぎる

## 2コマ目：先輩が暴走する

### 役割

- 笑いのピークを作る
- 感情で買いたくなる人間らしさを見せる
- 読者に「分かるけど危ない」と思わせる

### よく使う演出

- 集中線
- 爆発背景
- 紙吹雪
- キラキラ
- 祝勝会妄想
- 拳を上げる
- 札束や馬券の妄想。ただし煽りすぎない
- 分析者の冷や汗

### セリフの型

- 先輩「今日は勝ったな！！」
- 先輩「単勝！複勝！祝勝会！！」
- 先輩「未来が見えてる！！」
- 先輩「ここで勝負だ！！」
- 分析者「話が早すぎます！！」
- 分析者「まだ買い方を決めてません！！」

### NG例

- 全財産、借金、生活費など危険すぎる表現を軽く扱う
- ギャンブル煽りだけで終わる
- 分析者が何も止めない
- 先輩の暴走が弱すぎて笑いにならない

## 3コマ目：カナロア君が本質を指摘する

### 役割

- 作品の学びを置く
- 期待値と感情を切り分ける
- 資金管理や継続の視点へ戻す
- 先輩の暴走にブレーキをかける

### よく使う演出

- カナロア君が中央
- タブレットやホログラム
- 冷静な青系背景
- 先輩がピタッと止まる
- 分析者が「そう、それです」という顔
- 温度差で笑いを作る

### セリフの型

- カナロア君「A判定は『期待値が高い』という意味です。」
- カナロア君「全力投資していい、ではありません。」
- カナロア君「期待値と感情は別です。」
- カナロア君「期待値と資金管理は別です。」
- カナロア君「買う前に、金額を決めましょう。」
- 先輩「……」

### NG例

- 説明が長すぎる
- 専門用語だけで読者を置いていく
- カナロア君が冷たく見える
- 先輩を強く否定しすぎる
- 学びが曖昧なまま終わる

## 4コマ目：現実的な行動・教訓で着地する

### 役割

- 先輩を現実的な行動に戻す
- 分析者に安心させる
- カナロア君の一言でシリーズの思想を残す
- 読後感をよくする

### よく使う演出

- 先輩がしょんぼり、照れ、納得
- 小額投資、見送り、次走チェックなど具体的行動
- 分析者が笑顔
- カナロア君が優しく締める
- 背景は穏やか、かわいい、少し温かい

### セリフの型

- 先輩「……じゃあ、単複500円ずつでいくか」
- 先輩「今日は見送りにするか……」
- 先輩「金額、先に決めるわ……」
- 分析者「それが一番いいです！」
- カナロア君「勝つことより、続けることです。」
- カナロア君「退屈なくらいでちょうどいいです。」

### NG例

- 先輩が完全に反省して別人になる
- 最後が説教だけで終わる
- 教訓が強すぎて笑いが消える
- 具体的な行動がない
- ギャンブルで大勝ちして終わる
