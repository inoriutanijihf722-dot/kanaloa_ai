# X投稿文作成プロンプトテンプレート

完成した4コマ漫画に添えるX投稿文を作成してください。

## 入力欄

```text
今回のテーマ：
4コマの要約：
伝えたい教訓：
使いたいキーワード：
```

## 必ず含める思想

- 期待値
- 資金管理
- 感情を抑える
- 続けること
- カナロアAI

## ハッシュタグ候補

- #投資競馬
- #競馬AI
- #カナロアAI

## 出力形式

短め、標準、少し熱量ありの3パターンで出力してください。

```text
短め：
本文：
ハッシュタグ：

標準：
本文：
ハッシュタグ：

少し熱量あり：
本文：
ハッシュタグ：
```

## 投稿文のルール

- 煽りすぎない
- 競馬を知らない人にも意味が伝わる
- 学びと笑いの両方を残す
- 画像を見たくなる一言にする
- ハッシュタグは多くしすぎない
- 断定的な勝利保証に見える表現は避ける

## 投稿文の例

```text
短め：
本文：
A判定が出ても、財布までAテンションにしない。
期待値と資金管理はセットです。

ハッシュタグ：
#投資競馬 #競馬AI #カナロアAI

標準：
本文：
AIがA判定を出すと、つい気持ちが前に出ます。
でも大事なのは、当てることだけではなく続けること。
カナロアAIは今日も財布を冷静に戻します。

ハッシュタグ：
#投資競馬 #競馬AI #カナロアAI

少し熱量あり：
本文：
期待値が高いレースほど、感情も一緒に走り出す。
でも、そこで資金管理を崩したら長く続けられません。
勝つことより、続けること。
カナロアAIのいちばん大事な考え方です。

ハッシュタグ：
#投資競馬 #競馬AI #カナロアAI
```
